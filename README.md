# Berni
Berni est un adorable robot conçu spécialement pour aider à nettoyer les plages en ramassant les mégots de cigarette. Doté d'un design compact et élégant, Berni se déplace avec agilité sur le sable, tout en étant capable de naviguer facilement entre les serviettes de plage et les obstacles.
